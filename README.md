# token-possibillity-game

다음 단어 확률 맞추기 — Higher or Lower 스타일 웹 게임 (Next.js)

GPT · Claude · Gemini의 **토큰 확률**(🅰️ + 🅱️ + 기타 = 100%)을 비교하며, 빈칸에 더 잘 맞는 단어를 고릅니다.

## 실행

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) 에서 플레이

## 기능

- 15개 문장 풀에서 매 게임 5문장 랜덤 출제
- 좌/우 이미지 비교 UI
- 모델별 토큰 확률 + 평균(최종) 표시

## 스택

Next.js · React · TypeScript · Tailwind CSS
